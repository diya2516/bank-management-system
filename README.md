# Bank Management System

## Description
The Bank Management System is a console-based application developed using Python and MySQL. The project allows users to perform basic banking operations such as creating an account, depositing money, withdrawing money, and viewing account details. Python is used for the application logic and MySQL is used to store and manage customer data.

## Technologies Used
- Python
- MySQL
- MySQL Connector for Python

## Features
- Create a new bank account
- Deposit money
- Withdraw money
- View account details
- Store account information in MySQL database
- Simple command-line interface

## Database Setup
1. Install MySQL on your system.
2. Create a database for the project.
3. Create the required tables for storing account information.
4. Update the MySQL username and password in the Python code if needed.

## How to Run the Project
1. Install Python.
2. Install MySQL.
3. Install MySQL connector for Python.
   pip install mysql-connector-python
4. Run the Python file.
   python bank_management_system.py
